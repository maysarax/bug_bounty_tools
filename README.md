# Installing

We recommend Install any penetration testing distributions Like [ Backbox , Kali Linux ] .

git clone https://github.com/maysarax/bug_bounty_tools.git

cd bug_bounty_tools

chmod +x bug_bounty_tools.sh


Script Must Run As Root
